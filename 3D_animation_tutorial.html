Quick Reference

×

Keyframe

A drawing or pose that defines the starting and ending points of any smooth transition.

Interpolation

The process where the computer calculates the frames between two keyframes.

Rigging

Creating a skeleton of bones and controls for a 3D model so it can be moved.

Skinning

Binding the 3D mesh to the rig (bones) so the mesh deforms when bones move.

FK (Forward Kinematics)

Moving a chain of bones from the parent down to the child (Shoulder → Elbow → Hand).

IK (Inverse Kinematics)

Moving the end of the chain, causing parents to follow (Hand moves → Elbow/Shoulder adjust).

Workflow

Step-by-Step

Phase 1: Mesh Prep

Use Ctrl + R for Loop Cuts. Apply modifiers.

Topology matters! Ensure your mesh has "clean quads" (4-sided polygons) especially at joints like elbows and knees. This allows for smooth bending without collapsing the mesh.

Phase 2: Rigging

Add Armature Shift + A. Extrude bones.

Think of rigging as building a digital skeleton. Name your bones correctly (e.g., Forearm.L, Forearm.R) so that Blender's X-axis mirror features work automatically.

Phase 3: Skinning

Bind Mesh to Armature with "Automatic Weights".

Skinning determines which vertices move with which bone. "Weight Painting" visualizes this influence with color: Red (100% influence) to Blue (0% influence).

Phase 4: Animation

Pose Mode. Insert Keyframe I.

Switch to the "Dope Sheet" or "Graph Editor" view to manage your keyframes. Use the principle of 'Pose-to-Pose' animation for better control over timing.

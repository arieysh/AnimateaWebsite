1. The Rendering Pipeline

From 3D Geometry to 2D Image

What is Rendering?

Rendering is the process of generating an image from a model or scene file using computer programs. This scene file contains geometry, viewpoint, texture, lighting, and shading information.

ðŸ‘‰ Model + Materials + Lights + Camera = Final Image

There are two primary approaches (click to highlight):

Rasterization

Loops through polygons to determine visibility and color. This is the standard for real-time applications like video games .

Ray Tracing

Loops through pixels to determine which object is visible by simulating light rays. Produces high realism but is computationally expensive .

Advanced Techniques

Modern rendering often combines several advanced techniques to achieve photorealism:

  • Global Illumination (GI): Calculates both direct illumination (from lights) and indirect illumination (bounces).
  • Radiosity: A form of GI focusing on diffuse interreflection. Excellent for soft shadows and "color bleeding".
  • Image-Based Lighting (IBL): Uses High-Dynamic-Range (HDR) images of the real world to illuminate synthetic objects.
  • Photon Mapping: Simulates complex light paths, including caustics (focused light through glass).

2. Lighting the Scene

Setting the Mood and Revealing Depth

Types of Light Sources

Light Type Description Best Use Case
Point Light (Omni) Radiates light in all directions from a single point. Light bulbs, candles, general fill.
Directional Parallel rays from a specific direction (infinite distance). Sunlight or moonlight.
Spotlight Shines in a specific direction within a cone shape. Flashlights, stage lights, headlights.
Area Light Emits light from a specific boundary shape/size. Soft studio lighting.
